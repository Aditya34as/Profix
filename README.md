# Pro Fix - Home Service Platform

Pro Fix is a web-based platform that provides home services such as AC repair, geyser repair, and plumbing. Users can easily request a service, and a technician is assigned based on the request.

---

## Features

* Book home services online
* Contact form to request a technician
* AC repair service
* Geyser repair service
* Plumbing service
* Simple and user-friendly interface
* Responsive design

---

## How It Works

* User selects a service from the website
* User fills out the contact form with required details
* The request is sent to the server
* A technician is dispatched based on the request

---

## Installation

Clone the repository:

```id="k2d8x1"
git clone https://github.com/Aditya34as/Profix.git
cd Profix
```

Install dependencies:

Backend:

```id="z9m1q4"
cd backend
npm install
```

Frontend:

```id="t6n5r8"
cd frontend
npm install
```

Start backend server:

```id="p4v7s2"

```

Start frontend:

```id="y8c3l0"

```

